1. Code Change: Enable "Continuous" Mode
If you are using the browser's built-in API, you must explicitly tell it not to stop after one sentence.

JavaScript
const recognition = new window.webkitSpeechRecognition();

// This is the key! Set this to true.
recognition.continuous = true; 

// This allows the app to show text as he speaks
recognition.interimResults = true;
